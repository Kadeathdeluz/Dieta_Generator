# Dieta_Generator
Alea diet generator.
A random alea diet generator.
As internal data, the App contains 4 different diets (1 per week).
The App randomizes, mixes and shows that diets in one week.

Diet example:
Monday: Diet 1
Tuesday: Diet 3
Wednesday: Diet 1
Thursday: Diet 2
Friday: Diet 4
Saturday: Diet 1
Sunday: Diet 3

Generated diet has the following structure:

Day of the week: -
Starter: -
Main course: - 
Dinner: -
